# Toolshop API Testing

## Project Overview

This project demonstrates API testing using Postman, Newman, and GitHub.

## Tools Used

- Postman
- Newman
- GitHub
- JavaScript (Postman Tests)

## Project Files

- Toolsman.json - Postman Collection
- Lanre Newman Summary Report.pdf - Newman execution summary
- Lanre Postman Report 2.pdf - HTML report converted to PDF
- newman-run-report.html - Detailed Newman HTML report

## Test Coverage

The collection validates:

- GET Requests
- POST Requests
- PUT Requests
- DELETE Requests
- Response Status Codes
- Response Time
- JSON Schema Validation
- Automated Assertions

## How to Run

```bash
newman run Toolsman.json
```
